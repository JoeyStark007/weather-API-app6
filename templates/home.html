Weather Data API

URL format: http://127.0.0.1:5000/api/v1/station/date(yyyy-mm-dd)

URL example for a specific date and station: http://127.0.0.1:5000/api/v1/10/2000-11-05

URL format for a specific station for all dates: http://127.0.0.1:5000/api/v1/station
URL example for a specific station for all dates: http://127.0.0.1:5000/api/v1/10

URL format for a specific station for all the days in a year: http://127.0.0.1:5000/api/v1/yearly/station/year
URL example for a specific station for all the days in a year : http://127.0.0.1:5000/api/v1/yearly/3/2001
